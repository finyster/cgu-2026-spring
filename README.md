# CGU 2026 Spring Homework Repository

This repository is used to manage coursework and project files for the Spring 2026 semester.

## Structure

```text
homeworks/
├── Derivatives_and_Risk_Management/
└── Investment_Analysis_and_Portfolio_Management/
    └── hw2/
```

## Courses

### Derivatives and Risk Management

Course materials and projects for the derivatives and risk management course.

### Investment Analysis and Portfolio Management

Current contents:

- `hw2/`: asset pricing and portfolio management assignment

The `hw2` project includes:

- a reproducible Python analysis script
- raw and processed data snapshots
- regression outputs
- figures
- a Markdown report

## Notes

- Python virtual environments and cache files are excluded via `.gitignore`.
- Assignment outputs are intentionally kept in the repository for easier review and submission.
